# MyNews App 🚀
Developed using Android Studio Quail 1 2026.1.1 Patch 2, with concept:
- MVVM + Clean Architecture
- Multi Module
- Jetpack Compose
- Dagger Hilt (Dependency Injection)
- Retrofit + Okhttp
- Room Database (for caching to increase UX when bad internet connection)
- Reactive Programming

## 🚀 Getting Started
1. Clone the Repository
2. put your API Key from https://newsapi.org to local.properties as NEWS_API_KEY (for testing purpose, you can use my NEWS_API_KEY= d62444a741a240d7b142764e2e4b8211 )
